# CLI를 통한 아마존 마켓 서비스 구현

### 주요기능

  1. 회원가입 기능
  2. 로그인 로그아웃 기능
  3. 구매가능한 아이템의 목록이 표시되고 구매를 진행한다.
  (자세한 구현 기능은 동영상을 참고하세요.)
